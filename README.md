## Interactive Features

- `Ctrl+K` global search across all sections
- Copy-to-clipboard on all code blocks and prompts
- Category filtering on commands and prompts
- Python/TypeScript tab switching on code examples
- Token calculator with batch API toggle
- Collapsible sections for dense reference content
- Mobile hamburger nav
- Scroll-triggered reveal animations
- Konami code easter egg
- Session-stored intro (plays once, skips on return)

## Tech Stack

- Pure HTML, CSS, JavaScript
- Zero external dependencies (fonts from Google Fonts only)
- Single `index.html` file
- No build step, no bundler, no framework
- Works on any static file server

## Run Locally

```bash
# Any static server works
npx serve .
# or
python -m http.server 8000
# or just open index.html in a browser
```

## Deploy

Drop `index.html` on any web server. It's one file.
